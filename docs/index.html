Zero-Cost Abstractions

Emotions, styles, and SSML compile into a control graph before runtime. The engine stays fast and unaware of high-level concepts.

Swappable Backends

Switch between Kokoro (GPU optimized), Piper (CPU fast), or your own custom TTS engine without changing application code.

Streaming Ready

Built for real-time applications. Stream audio sentence-by-sentence with sentence boundary awareness.
